1.1- Compilação do código:

através das ferramentas oferecidas pelo software. A ferramenta de compilação
se encontra no menu Executar -> Compilar. Após compilado você pode utilizar

2- Utilização dos programas:



2.1- Par mais próximo:



2.1.1 - Funcionamento programa: 
O programa "Par mais próximo" é um programa em c que tem como objetivo
encontrar o par de números com a menor diferença, em módulo, entre eles,
ou seja, os "mais próximos".

2.1.2 - Entradas do programa:
As entradas desse programa devem ser somente números contidos no conjunto 
dos inteiros. A primeira entrada corresponde ao número de elementos com que
o programa irá trabalhar. As entradas seguintes correspondem aos elementos 
em si, LEMBRE-SE: Os valores inseridos devem ser inteiros!

2.1.3 - Saída do programa:
O programa devolverá como saída o par que fornece a menor diferença assim
como qual é o valor dessa diferença.



2.2 - Elemento único:



2.2.1 - Funcionamento do programa:
O programa "Elemento único" é um programa em c que tem como objetivo
encontrar os elementos que aprecem uma única vez em um determinado
conjunto de números inteiros dados.

2.2.2 - Entradas do programa:
As entradas desse programa devem ser somente números contidos no conjunto 
dos inteiros. A primeira entrada corresponde ao número de elementos com que
o programa irá trabalhar. As entradas seguintes correspondem aos elementos 
em si, LEMBRE-SE: Os valores inseridos devem ser inteiros!

2.2.3 - Saída do programa:
O programa devolverá como saída os elementos que aparecem uma única. Caso
não exista nenhum ele não devolverá nada.



2.3 - Distribuição de frequência:



2.3.1 - Funcionamento do programa:
O programa "Distribuição de frequência" é um programa em c que tem como 
objetivo indicar os elementos que ocorrem com maior frequência em um dado
conjunto de números inteiros.

2.3.2 -  Entradas do programa:
As entradas desse programa devem ser somente números contidos no conjunto 
dos inteiros. A primeira entrada corresponde ao número de elementos com que
o programa irá trabalhar. As entradas seguintes correspondem aos elementos 
em si, LEMBRE-SE: Os valores inseridos devem ser inteiros!

2.3.3 - Saída do programa:
O programa devolverá como saída todos os elementos que ocorrem com maior 
frequência assim como o número de vezes que eles ocorrem.
